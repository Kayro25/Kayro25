## Projects

### [autoB](https://github.com/Kayro25/autoB)
> A **modern CLI** for scaffolding projects from reusable YAML templates.
✔ Interactive file-tree navigation for destination selection.
✔ Template variables with runtime prompts.
✔ Post-creation commands (e.g. `npm install`, `git init`).
✔ Available on npm: `npm install -g @kayro25/autob`

---

### [scrapCrypto](https://github.com/Kayro25/scrapCrypto)
> A **Galxe quest ranker & alert bot** powered by Python.
✔ Fetches live campaigns via the Galxe GraphQL API.
✔ Scores quests by reward, chain, and type heuristics.
✔ Exports results to JSON + HTML ranking table.
✔ Sends Telegram alerts for high-value quests.

---

### [Cortesana](https://github.com/Kayro25/cortesana)
> A **retro terminal music player** built with Python.
✔ Interactive TUI using `urwid` — no GUI required.
✔ Play/Pause, Next/Prev, Shuffle & Loop controls.
✔ Spectrum visualizer with `numpy` + `matplotlib`.
✔ Reads MP3 metadata and album art via `mutagen`.

---

### [Console-DungeonCrawler](https://github.com/Kayro25/Console-DungeonCrawler)
> A **turn-based dungeon crawler** that runs entirely in the terminal.
✔ Written in **C#** with classic RPG mechanics.
✔ Procedural dungeon exploration via the console.
